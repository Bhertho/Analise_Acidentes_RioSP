# Análise Exploratória

Esse repositório possui dois propósitos:
1. Realizar uma análise exploratória com python de uma base de dados de acidentes ocorridos nos trechos de rodovias concedidos a concessionária RioSP.
2. Praticar e aprender a utilizar gráficos com seaborn e matplotlib.

Então o objetivo final desse projeto é aprender a usar as bibliotecas gráficas do python, ao mesmo tempo que faz uma análise de uma base de dados real.

Por ser para fins de aprendizado, o notebook estará bastante comentado e bem estruturado.
